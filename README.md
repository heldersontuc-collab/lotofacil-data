# Lotofácil – Base de Dados Oficial

Este repositório contém o histórico completo dos concursos da Lotofácil em formato CSV.

## Estrutura
- `data/lotofacil.csv` — Histórico completo dos concursos
- Separador: vírgula `,`
- Colunas:
  - concurso
  - data (YYYY-MM-DD)
  - d1 até d15

## Uso direto (RAW)
O arquivo pode ser consumido diretamente via URL RAW do GitHub.

Exemplo:
https://raw.githubusercontent.com/heldersontuc-collab/lotofacil-dados/main/data/lotofacil.csv
